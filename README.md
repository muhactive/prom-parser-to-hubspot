# prom-parser-to-hubspot
Python scraper for Prom.ua with Google Sheets export and HubSpot CRM sync
# Prom.ua Parser → Google Sheets → HubSpot CRM

Python проект для парсинга товаров с Prom.ua с последующей отправкой в Google Sheets и HubSpot CRM.

## Возможности

- Парсинг всех товаров в категории Prom.ua
- Сбор данных: название, цена, наличие, фото, ссылка
- Экспорт данных в Google Sheets
- Синхронизация данных в HubSpot CRM через API

```bash
